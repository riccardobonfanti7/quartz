## Cosa Sono i Metadati?
I metadati sono dati che descrivono le caratteristiche di un file, come il nome, la dimensione, la data di creazione e modifica, i permessi, e cos√¨ via.

## Visualizzare i Metadati
Puoi visualizzare i metadati di un file utilizzando il comando `ls -l` o `stat`.

### Esempi
```bash
# Visualizza i metadati dettagliati di un file
ls -l nomefile.txt

# Visualizza metadati estesi
stat nomefile.txt
